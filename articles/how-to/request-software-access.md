# Request Access to Approved Software

## Summary

Use this article to request access to approved company software.

## Audience

Employees who need access to a software tool for their job.

## Before You Start

You need:

- Your company login
- The name of the software
- A business reason for the request
- Manager approval, if required

## Steps

### 1. Open the service portal

Go to the internal service portal.

### 2. Select Software Access

From the main menu, select:

```text
Requests > Software Access
