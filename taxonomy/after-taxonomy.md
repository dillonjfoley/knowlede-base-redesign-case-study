# After Taxonomy

The redesigned knowledge base is organized by task and user need.

| Top-Level Category | Subcategory | Example Article |
|---|---|---|
| Access and Passwords | Password Help | Reset your network password |
| Access and Passwords | Multi-Factor Authentication | Set up multi-factor authentication |
| Network and VPN | VPN Setup | Connect to VPN for the first time |
| Network and VPN | VPN Troubleshooting | Fix VPN connection errors |
| Software Requests | Request Software | Request access to approved software |
| Hardware and Devices | Laptop Support | Prepare your laptop for remote work |
| Email and Calendar | Email Troubleshooting | Fix Outlook sync issues |
| File Access and Storage | Shared Folder Access | Request access to a shared folder |

## Why This Works Better

- Users can browse by task.
- Search terms match common user language.
- Troubleshooting content is easier to separate from how-to content.
- Content owners can manage smaller article groups.
- Duplicate content is easier to find and remove.
