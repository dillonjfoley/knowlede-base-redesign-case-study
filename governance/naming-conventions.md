# Naming Conventions

These naming conventions keep the knowledge base clear, searchable, and consistent.

## Article Titles

Use action-based titles.

Good:

- Reset your network password
- Request access to approved software
- Fix VPN connection errors
- Connect to Wi-Fi on a company laptop

Avoid:

- VPN
- Password Info
- Software
- Outlook Problems
- Identity Manager Access Process

## Title Patterns

| Article Type | Pattern | Example |
|---|---|---|
| How-to | Verb + task | Request access to approved software |
| Troubleshooting | Fix + issue | Fix VPN connection errors |
| Reference | Noun + overview | Laptop replacement policy |
| FAQ | Question format | Why did my password expire? |

## File Names

Use lowercase letters and hyphens.

Good:

```text
request-software-access.md
fix-vpn-connection-errors.md
reset-network-password.md
